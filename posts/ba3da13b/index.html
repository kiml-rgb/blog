使用 WSL

启动和退出 Linux

进入 Linux 子系统有以下几种方式：

（1）和正常软件一样启动，如在开始菜单中点击其图标、双击桌面快捷方式
（2）在 Windows 中打开一个终端，如 PowerShell，输入命令 wsl
（3）在终端中输入命令 Linux 发行版名称.exe，如 ubuntu20.04.exe

退出 Linux 子系统的方式：

（1）直接关闭终端
（2）在终端中输入命令 exit，此时退出了 WSL，然后输入命令 wsl -t Ubuntu-20.04，关闭 ubuntu 子系统。
