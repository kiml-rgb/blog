安装

之前在写 [[Obsidian + Git + Hexo + 图床 = Blog + 个人笔记体系]]这篇笔记的时候有提到过Git的安装。这里就直接把步骤贴过来，具体安装地址(https://git-scm.com/)

  1. 访问 git 官方地址，下载对应的安装包，进行安装（简单的点击下一步）。

  2. 安装好之后，鼠标右键可以看到：Git Bash Here，点击后打开了。

  3. 输入：git --version
    出现版本号说明安装成功。
